======================================================
 Regular HTML/CSS/JS
======================================================
1. Add the usps-bootstrap folder into your assets directory of your project.
2. On your HTML page, include ALL files under usps-bootstrap/dist
*NOTE* File order matters: Bootstrap CSS first, then USPS Styles, then your app's CSS.

Example:
		<link href="assets/usps-bootstrap/dist/usps.bootstrap-4.0.0.css" rel="stylesheet">
		<link href="assets/usps-bootstrap/dist/usps.styles.css" rel="stylesheet">

3. On your HTML page, include ALL JS dependency libraries under usps-bootstrap/lib.
*NOTE* File order matters: JQuery first, then Popper, then Bootstrap JS, then your app's JS.

Example:
		<script src="assets/usps-bootstrap/lib/jquery-3.2.1.min.js">
		<script src="assets/usps-bootstrap/lib/popper.min.js">
		<script src="assets/usps-bootstrap/lib/bootstrap-4.0.0.js">

Review documentation for usage.
https://getbootstrap.com/

======================================================
 Angular
======================================================
1. Add the usps-bootstrap folder into your assets directory of your project.
2. On angular.json, include ALL files under the usps-bootstrap/dist
*NOTE* File order matters: Bootstrap CSS first, then USPS Styles, then your app's CSS.

Example:
            "styles": [
              "src/styles.css",
              "src/assets/usps-bootstrap/dist/usps.bootstra-4.0.0.css",
              "src/assets/usps-bootstrap/dist/usps.styles.css"
            ],
			
3. Install new bootstrap package: npm install ngx-bootstrap
*NOTE* ngx-bootstrap removes the JS dependencies and provide angular approach components.

Review documentation for usage.
https://getbootstrap.com/
https://valor-software.com/ngx-bootstrap/#/
